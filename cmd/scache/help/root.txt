SCache is a smart Go struct cache code generator that automatically scans structs and generates cache methods.

Commands:
  gen     Generate cache code (default)
  version Show version info
  help    Show help

Quick Start:
  scache gen -g                     # Generate generic version (recommended)
  scache gen -dir ./models          # Specify directory
  scache gen -structs User,Product  # Specific structs

More Help:
  scache gen --help                 # Detailed gen options
  scache [command] --help           # Command help
