## 关键决策
1. **Sync deletion over async**: 使用同步删除替代异步，避免竞态条件
2. **noopPolicy for unlimited**: MaxSize <= 0 时返回 noopPolicy{} 而不是默认容量 100
3. **Unified tests directory**: 所有测试统一放在 `tests/` 目录
4. **Embedded version**: 使用 `//go:embed version.info` 在编译时嵌入版本号
5. **Template API**: 使用 `*config.EngineConfig` 参数，nil 使用默认配置
6. **Error format**: 错误消息小写开头，无标点结尾
7. **Skip generated files**: 自动跳过 scache 生成的文件
